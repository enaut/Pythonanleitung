Natürlich kann `while` auch mit `turtle` zusammen verwendet werden.

```python
import turtle
t = turtle.Pen()
while 1 == 1:
    t.forward(100)
    t.left(60)
```

![Turtlesechseck](img/Turtlewiederholungenwhile.png)

Das Turtle läuft immer weiter auf der selben Spur.
